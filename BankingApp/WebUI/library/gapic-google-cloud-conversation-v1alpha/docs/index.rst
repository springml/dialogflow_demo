GAPIC library for the Cloud Conversation Engine API
=============================================================================================================

This is the API documentation for ``gapic-google-cloud-conversation-v1alpha``.

gapic-google-cloud-conversation-v1alpha uses google-gax_ (Google API extensions) to provide an
easy-to-use client library for the `Cloud Conversation Engine API`_ (v1alpha) defined in the googleapis_ git repository


.. _`google-gax`: https://github.com/googleapis/gax-python
.. _`googleapis`: https://github.com/googleapis/googleapis/tree/master/google/cloud/conversation/v1alpha
.. _`Cloud Conversation Engine API`: https://developers.google.com/apis-explorer/?hl=en_US#p/conversation/v1alpha/
